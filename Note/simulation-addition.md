# 🧮 模拟竖式加法 (Simulation Addition) 学习笔记

## 1. 核心场景
当我们需要处理“大数加法”或者特定进位逻辑（如二进制加法）时，直接使用 `Number` 会导致精度溢出。此时需要模拟我们在纸上计算竖式的过程。

## 2. 通用代码模板 (The "Golden Template")
这套模板可以通用在字符串相加、二进制求和、数组形式加法等题目中：

```typescript
function add(p1: string | number[], p2: string | number[]): any {
    let res = [];
    let carry = 0; // 进位
    let i = p1.length - 1;
    let j = p2.length - 1;

    // 只要还有位没算完，或者还有进位，就继续
    while (i >= 0 || j >= 0 || carry > 0) {
        // 1. 获取当前位的值（指针越界则补 0）
        const x = i >= 0 ? Number(p1[i]) : 0;
        const y = j >= 0 ? Number(p2[j]) : 0;

        // 2. 计算当前位的总和
        const sum = x + y + carry;

        // 3. 更新结果位和新进位 (10 进制为例)
        res.unshift(sum % 10);      // 当前位
        carry = Math.floor(sum / 10); // 进位

        // 4. 指针前移
        i--;
        j--;
    }
    return res;
}
```

## 3. 核心技巧与避坑
- **指针与边界**：使用 `i >= 0 || j >= 0 || carry > 0` 一个循环搞定所有位，不再需要分段处理。
- **补零思想**：`i >= 0 ? val : 0` 优雅地解决了两个数字长度不等的问题。
- **头部插入**：字符串可用 `res = bit + res`，数组可用 `unshift()`，这样最后不需要 `reverse()`。
- **进制切换**：
  - 二进制：`sum % 2` 和 `sum / 2`
  - 十进制：`sum % 10` 和 `sum / 10`
