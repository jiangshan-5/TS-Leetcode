# 🧠 动态规划 (Dynamic Programming) 学习笔记

## 1. 什么是动态规划？
动态规划（DP）是“**记住过去，推导未来**”的算法。它将复杂问题拆分为重叠的子问题，通过存储子问题的解（“填表”）来避免重复计算。

## 2. DP 核心五步法 (手写代码逻辑)
1.  **确定 dp 数组及下标含义**：
    - 例如：`dp[i]` 是爬到第 `i` 阶的方法数。
2.  **确定递推公式（状态转移方程）**：
    - 规律：`dp[i] = dp[i-1] + dp[i-2]`。
3.  **dp 数组初始化**：
    - 确定基础边界值，如 `dp[1] = 1`, `dp[2] = 2`。
4.  **确定遍历顺序**：
    - 通常是自底向上（从 0 到 n 的循环）。
5.  **举例推导验证**：
    - 模拟 `i=1, 2, 3` 的情况，看结果是否符合预期。

## 3. 开发避坑指南（TS/JS 实战经验）
- **【报错核心】二维数组初始化**：
  在操作 `dp[i][j]` 前，必须先确保 `dp[i]` 已经被初始化为数组。
  ```typescript
  for (let i = 2; i <= rowIndex; i++) {
      dp[i] = []; // 核心：先初始化当前行，再进行内部赋值
      for (let j = 0; j <= i; j++) {
          dp[i][j] = ...
      }
  }
  ```
- **越界问题**：内层循环的 `j` 范围要严格对应当前行的长度（通常是 `j <= i`）。
