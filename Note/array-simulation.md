# 🛠️ 数组与模拟 (Array & Simulation) 学习笔记

## 1. 双指针法 (Two Pointers)
双指针是处理数组问题最常用的技巧之一，通常可以大幅降低时间复杂度。

### 核心应用场景
- **快慢指针**：用于移动元素、删除重复项（如 283. 移动零）。
- **对撞指针**：用于反转字符串、寻找两数之和。

### 实战：移动零 [283]
**思路**：`fast` 遍历，`slow` 记录下一个非零放置位置。
```typescript
function moveZeroes(nums: number[]): void {
    let slow = 0;
    for (let fast = 0; fast < nums.length; fast++) {
        if (nums[fast] !== 0) {
            [nums[slow], nums[fast]] = [nums[fast], nums[slow]];
            slow++;
        }
    }
}
```

### 实战：三数之和 [15]
**思路**：排序 + 对撞指针。
1. **排序**：升序排序是去重和使用双指针的前提。
2. **固定一个数**：遍历数组，固定 `nums[i]`。
3. **对撞指针**：在 `(i, n-1]` 范围内使用 `L` 和 `R` 指针，根据 `sum` 与 0 的关系移动。
4. **去重核心**：
   - `i` 去重：`if (i > 0 && nums[i] === nums[i-1]) continue;`
   - `L` 和 `R` 去重：在找到解后，通过 `while` 循环跳过所有相同的元素。

### 实战：字符串压缩 [面试题 01.06]
**思路**：快慢指针（分组循环）。
1. `slow` 指向每组字符的开始。
2. `fast` 探路，直到遇到不同字符。
3. `fast - slow` 即为组长度。
4. 这种“分组循环”的结构比单指针+计数器更稳健，不需要处理繁琐的边界条件。

## 2. 矩阵旋转与模拟 [1861]
旋转盒子结合了“重力模拟”与“矩阵旋转”。

### 解题核心
1. **重力模拟**（行内位移）：从后往前遍历，维护一个 `k` 指针指向最右侧可放置的位置。
2. **矩阵旋转 90 度**：
   - 坐标映射公式：`newGrid[j][m - 1 - i] = oldGrid[i][j]`
   - 其中 `m` 是原矩阵的行数。
